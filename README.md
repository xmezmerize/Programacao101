# Programação101 - UFMG

O curso foi feito em 2026/01 e aborda o percurso denominado de *Touring Completo*, onde o aluno será capaz de usar todas as principais estruturas de uma linguagem como (funções, coleções, estruturas de controle e repetição, dados primitivos, variáveis, classes, métodos, bibliotecas, etc).

Materiais e sites usados:

- [PyPi](https://pypi.org/) ⮕ site de suporte para instalação e uso de bibliotecas externas

- [PythonTutor](https://pythontutor.com/) ⮕ site _step by step_ para visualização de código

- [Colab](https://colab.research.google.com/) ⮕ IDE online python do google
